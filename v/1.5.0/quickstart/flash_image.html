Installing a pre-built system imageÔÉÅ

To install a pre-built image on a Synaptics Astra evaluation kit:

  1. Download a prebuilt image suitable for your development kit from GitHub

  2. Follow the instructions in Updating the Firmware to install the image on the board

  3. Connect to the board using the instructions in Linux OS Login
